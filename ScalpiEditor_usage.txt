This is ScalpiEditor - "heirarhy" text editor.

for open existing file just write file name as argument:
$ ScalpiEditor "~/.config/ScalpiEditor/binds"

key bindings:
exit                   [Ctrl] + [q]
save                   [Ctrl] + [s]

[navigation]
up                     [Up]
down                   [Down]
left                   [Left]
right                  [Right]
enter  block           [Enter]
escape block           [Esc]
go to left   word      [Ctrl]  + [←]
go to right  word      [Ctrl]  + [→]
go to start  line      [Ctrl]  + [1]
go to end    line      [Ctrl]  + [4]
go to line from number [Ctrl]  + [g]

[edit]
insert char            [any]
delete prev char       [BackSpace]
delete next char       [Del]
create upper line      [Shift] + [Enter]
create below line      [Alt]   + [Enter]
cut                    [Ctrl]  + [x]
copy                   [Ctrl]  + [c]
paste                  [Ctrl]  + [v]
paste inside block     [Alt]   + [Ctrl] + [v]
swap with upper  line  [alt]   + [↑]
swap with bottom line  [alt]   + [↓]
swap to commentary     [Alt]   + [Slash]       

[marks]
create new mark        [Ctrl]  + [slash]
jump to prev mark      [Ctrl]  + [↑]
jump to next mark      [Ctrl]  + [↓]

[selection]
set block selected     [Ctrl]  + [a]
select left  symbol    [Shift] + [←]
select right symbol    [Shift] + [→]
